# Creep and Collect

Creep and Collect was made as an entry to Hack Club -> Athena -> Sleepover -> SnoozeFest game jam. This game was made using godot and published on itch.io. Play it [here](https://hack-girl.itch.io/creep-and-collect): https://hack-girl.itch.io/creep-and-collect

![Game Screenshot](images\GoGame.png)


## Aim of the game
You need to collect all bananas, without waking the snails up (by bumping into them) to win.

## How to Play

- Move left and right with A/left arrow and D/right arrow respectively
- Jump with W / Enter key / Space bar / up arrow key
- Bump into bananas to collect them
- If you wake up the snails, you loose
- Collect all bananas to win the game
